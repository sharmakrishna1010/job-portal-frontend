# Job Portal Frontend

This repository contains the frontend code for a full-stack job portal application. This project was built by following a tutorial to integrate a React frontend with a Spring Boot backend. 

> **Note:** This repository only contains the frontend. A separate Spring Boot backend is required for full functionality.

## Tech Stack

Based on the project configuration, this frontend is built using:

* **Framework:** React
* **Language:** TypeScript
* **Build Tool:** Vite
* **Linting:** ESLint
